# Essence Dialog - Dialog component

Options:
- visible: boolean value with `true` for visible and `false` for hidden

### How to use
```js
import {Dialog, DialogHeader, DialogContent, DialogFooter} from 'essence-dialog';

<Dialog visible={true}>
	<DialogHeader>
		Dialog header
	</DialogHeader>
	<DialogContent>
		Dialog content
	</DialogContent>
	<DialogFooter>
		Dialog footer
	</DialogFooter>
</Dialog>
```
