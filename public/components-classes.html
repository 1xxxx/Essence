Useful Classes

According to our needs we created some helper classes and stored them in a .less file called utils.
You cand use them, and create other classes to suit your own needs. Here they are:

  • .e-left - use it to float your element to the left

  • .e-right - use it to float your element to the right

  • .e-padding-top-25 - useful whenever you need to add padding to your elements. The padding value can be: 15, 25, 50, 100, 150, 200, 250.

  • .e-padding-bottom-25 - useful whenever you need to add padding to your elements. The padding value can be: 15, 25, 50, 100, 150, 200, 250.

  • .e-margin-top-25 - useful whenever you need to add oustide margin to your elements. The margin value can be: 15, 25, 50, 100, 150, 200, 250.

  • .e-margin-bottom-25 - useful whenever you need to add oustide margin to your elements. The margin value can be: 15, 25, 50, 100, 150, 200, 250.

  • There are also the classes that don't need any explanation: .e-no-padding and .e-no-margin.

  • If you don't want to display an element to mobile phones, just add te .e-no-mobile class. The same applies to tablets in portrait mode with the.e-no-tablet class. Note that the element hidden from tablet is also hidden from mobiles.

  • To control just the line-height of a text, use the helper clases .line-height-11, .line-height-12, .line-height-13, .line-height-14, .line-height-15. They set the line-height to 1.1, 1.2, 1.3, 1.4 and 1.5 respectively.

  • If you want to reset the looks of an unordered or ordered list, add the .e-no-style class. If you want al your <li>'s to have a border-bottom, except the last one, use the .e-li-border-bottom class on the list element.

  • Images are best displayed with the .e-img-rsp class. It makes the picture resize according to it's parent container.
