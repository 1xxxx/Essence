Lists

Lists are best suited to presenting a homogeneous data type or sets of data types, such as images and text. Take a look at the examples below and pick one - and keep it consistent as much as possible.

How to use

Use the "List" tag to render a group or single "ListItem" component. The "List" React tag will return a div container with a single or multiple <ListItem> React tags with the class name "e-list".

• Structure 

<List>
  <ListItem>
    __li-style-block-or-react-components__
  </ListItem>
</List>
• Styling
- classes: add extra class names to List or ListItem component from Styles > Colors menu

• Options
- type: single-line (for a single line text), two-line (for a two lines text) or multi-line (for 3 lines text)
- use the examples from above to structure your own List
