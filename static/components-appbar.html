How to use

Use the "AppBar" tag to render a div html tag with "e-appbar" classname.
Inside the AppBar component you can include any type of components. The default structure is (from left to right):

  1. Icon or Button component
  2. Menu component with MenuItem subcomponent
  3. Button component (Search button)
  4. Button component (Favorit button)
  5. Menu component with Icon option as indication
