Get Started with Essence

Learn how to use Essence for your application

Install

Essence is available as a npm package. For dependency management and JSX transformations, you can use browserify and reactify. Since the CSS framework is written in Less, you'll need to compile that as well.

React-Tap-Event Plugin

A few components use react-tap-event-plugin to listen for touch events. This temporary dependency exists since react v1.0 was not yet released. Once that happens, this plugin will no longer be required. In the meantime make sure to inject this plugin at the start of your app.

Roboto

Don't forget to include the Roboto and Lato fonts in your project. Visit Google Fonts in order to learn how to do that.
