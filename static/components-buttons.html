Buttons

There are 3 types of material buttons: FAB (floating action button), Raised and Flat.

Button FAB

We believe FAB stands for fabulous. Here’s a fabulous class for it: .e-btn-fab. The small one has the "-mini" addition to the class.

HTML CODE

<button class="e-btn-fab">
  <i class="e-icon-action-favorite"></i>
</button>
<button class="e-btn-fab-mini">
  <i class="e-icon-action-favorite"></i>
</button>

REACT CODE

<Btn>
  <BtnItem
    icon='action-favorite'
    type='fab'
    rippleEffect={true}
  />
  <BtnItem
    icon='action-favorite'
    type='fab'
    rippleEffect={true}
    mini={true}
  />
</Btn>

Button Raised

The main colors of buttons .default .primary .succes .info .warning .danger can be used for all buttons.

Buttons are also fabulous, whether they’re raised or flat. We’ve created several button types, which can be used by adding a button class - .raised or .flat.

HTML CODE

<button class="e-btn-default raised">
  default
</button>
<button class="e-btn-primary raised">
  primary
</button>
<button class="e-btn-succes raised">
  succes
</button>
<button class="e-btn-info raised">
  info
</button>
<button class="e-btn-warning raised">
  warning
</button>
<button class="e-btn-danger raised">
  danger
</button>

REACT CODE

<Btn>
  <BtnItem
    classes={'raised'}
    label='Label Default'
    tooltip='Tooltip Default'
    type='default'
    rippleEffect={true}
  />
  <BtnItem
    classes={'raised'}
    label='Label Primary'
    tooltip='Tooltip Primary'
    type='primary'
    rippleEffect={true}
  />
  <BtnItem
    classes={'raised'}
    label='Label Succes'
    tooltip='Tooltip Succes'
    type='succes'
    rippleEffect={true}
    disabled={true}
  />
  <BtnItem
    classes={'raised'}
    label='Label Info'
    tooltip='Tooltip Info'
    type='info'
    rippleEffect={true}
  />
  <BtnItem
    classes={'raised'}
    label='Label Warning'
    tooltip='Tooltip Warning'
    type='warning'
    rippleEffect={true}
  />
  <BtnItem
    classes={'raised'}
    label='Label Danger'
    tooltip='Tooltip Danger'
    type='danger'
    rippleEffect={true}
  />
</Btn>

Button Flat

If you want flat buttons, just add the .flat class to your button.

HTML CODE

<button class="e-btn-default flat">
  default
</button>
<button class="e-btn-primary flat">
  primary
</button>
<button class="e-btn-succes flat">
  succes
</button>
<button class="e-btn-info flat">
  info
</button>
<button class="e-btn-warning flat">

</button>

  danger
