Switches

Switches allow the user to select options. There are three kinds: checkboxes, radio buttons, and on/off switches. Here they are, in all their splendor.

HTML CODE

<div class="e-switch">
  <div class="e-radio e-radio-success">
    <label>
      <input type="radio" name="radioButton" value="value1"><span class="absolute circle"></span><span class="absolute e-check"></span><span>Radio Button 1</span>
    </label>
  </div>
</div>
<div class="e-switch">
  <div class="e-radio e-radio-success">
    <label>
      <input type="radio" name="radioButton" value="value2"><span class="absolute circle"></span><span class="absolute e-check"></span><span>Radio Button 2</span>
    </label>
  </div>
</div>
<div class="e-switch">
  <div class="e-checkbox">
    <label>
      <input type="checkbox" name="checkbox1" class="toggle"><span class="absolute e-wave"></span><span class="absolute e-check-valid"></span><span>Switch Checkbox</span>
    </label>
  </div>
</div>
<div class="e-switch">
  <div class="e-switches">
    <label><span>Switches before</span>
      <input type="checkbox"><span class="e-switches-toggle"></span><span>Switches after</span>
    </label>
  </div>
</div>

<Switch>
  <SwitchItem
    type='radio'

    name='radioButton'

  </SwitchItem>

  <SwitchItem
    type='radio'
    text='Radio Button 2'
    name='radioButton'
    defaultValue='value2'
  >
  </SwitchItem>

  <SwitchItem
    type='checkbox'
    text='Switch Checkbox'
    name='checkbox1'
  >
  </SwitchItem>

  <SwitchItem
    type='switches'
    beforeText='Switches before'
    afterText='Switches after'
    checked={false}
    name='switches1'
  >
  </SwitchItem>
</Switch>

How to use

Use the "Switch" tag to render a group of "SwitchItem" components. Inside the Switch component you can include only the SwitchItem component.
• Structure 

<Switch>
  <SwitchItem
  type='__radio-or-checkbox-or-switches'
  text='__your-text__'
  name='__input-name__'
  checked='__true-or-false__'
  beforeText='__preppend-your-text__'
  afterText='__append-your-text__'
  >
  </SwitchItem>
</Switch>
• Styling
- no styling required

• Options
- small - true or false with type: circle will render a small circle visualization - type
1. slider: appends "slider" to the class name "e-progress-" in the following format "e-btn-" + type
2. circle: appends "circle" to the class name "e-progress-" in the following format "e-btn-" + type
3. dots: appends "dots" to the class name "e-progress-" in the following format "e-btn-" + type
4. fix: appends "fix" to the class name "e-progress-" in the following format "e-btn-" + type
5. dynamic: appends "dynamic" to the class name "e-progress-" in the following format "e-btn-" + type
- defaultValue - set the default value of input
- checked - true or false set the checked attribute value
