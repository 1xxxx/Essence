How to use

Use the "Slider" tag to render a group of "SliderItem" components. The "Slider" React tag will return a div container with a group of <div> html tags for SliderItem components. Inside the Slider component you can include only the SliderItem component.
A "SliderItem" React component will render one <input > html tag for the current value to be set as the input default value.

• Options
- start: integer value to set the initial value; values: 0 to 100
- fill: return a colored track to indicate the current value position
- discrete: if set to true it uses the "discrete" class name to render a indicator for the current value
- disabled: if set to true it uses the "disabled" class name to render a non-editable component
