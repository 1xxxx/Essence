# Essence Toast - Toast component.

Options:
- visible: `true` - show the toast
- visible: `false` (**default**) - hide the toast
- delay: `2000` (**default**) - miliseconds delay to hide

### How to use
```
<Toast classes={'e-text-green-500'} visible={true} delay={2000}>
	<Block>Toast</Block>
</Toast>
```
